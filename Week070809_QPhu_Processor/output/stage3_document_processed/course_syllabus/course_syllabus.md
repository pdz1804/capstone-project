ADVANCED AI &amp; MACHINE LEARNING COURSE

HCMUT - Fall 2024

## COURSE INFORMATION:

Course Code: CS4011 Course Title: Advanced Artificial Intelligence and Machine Learning Credits: 3 Prerequisites: Linear Algebra, Probability &amp; Statistics, Programming

## INSTRUCTOR INFORMATION:

Professor: Dr. Nguyen Van AI Email: nguyen.ai@hcmut.edu.vn Office: Faculty of Computer Science, Room 408 Office Hours: Monday &amp; Wednesday, 2:00-4:00 PM

## COURSE DESCRIPTION:

This course provides an in-depth study of advanced artificial intelligence and machine learning techniques with emphasis on practical applications in real-world scenarios. Students will learn multimodal AI systems, deep learning architectures, and modern NLP techniques.

## LEARNING OBJECTIVES:

Upon successful completion of this course, students will be able to:

1. Understand and implement advanced deep learning architectures
2. Design multimodal AI systems for complex data processing
3. Apply state-of-the-art NLP techniques to real problems
4. Evaluate and optimize AI models for production environments
5. Integrate multiple AI technologies in comprehensive solutions

## COURSE SCHEDULE:

Week 1-2:   Introduction to Advanced AI Concepts - Neural Network Fundamentals Review - Modern Deep Learning Architectures - Practical: Setting up ML Development Environment

Week 3-4:   Computer Vision and Image Processing - Convolutional Neural Networks (CNNs) - Vision Transformers (ViTs) - Object Detection and Segmentation - Practical: Image Classification Project

Week 5-6:   Natural Language Processing - Transformer Architecture Deep Dive - BERT, GPT, and Modern Language Models - Text Classification and Generation - Practical: NLP Pipeline Development

Week 7-8:   Multimodal AI Systems - Cross-modal Learning - Document Processing and Analysis - Audio-Visual Integration - Practical: Multimodal Content Processor

Week 9-10:  Advanced Topics - Reinforcement Learning Applications - Graph Neural Networks - Federated Learning - Practical: Choose Your Adventure Project

Week 11-12: Production AI Systems - Model Deployment and Optimization - MLOps and Continuous Integration - Monitoring and Maintenance - Practical: Deploy to Production

Week 13-14: Research Frontiers - Current Research Trends - Student Presentations - Final Project Showcase
