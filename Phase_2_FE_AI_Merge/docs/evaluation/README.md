# Evaluation Guides

Feature guides for the evaluation runners in `backend/evals`.

- `retriever_e2e_evaluation_summary.md`: retrieval and E2E methodology.
- `chunking_evaluation_summary.md`: Chroma-style span-level chunking eval.
- `parsing_omnidocbench_summary.md`: PDF parsing eval on OmniDocBench.
- `parsing_officedocbench_summary.md`: Office parsing eval on OfficeDocBench.
- `media_frame_alignment_evaluation_summary.md`: media parsing and frame alignment.
- `parsing_bookmark_track_changes_summary.md`: parser improvements around bookmarks and track changes.

Benchmark data and generated result folders are local-only and should not be committed.
